# Annual Planner Template

> Drop this file into the planner app to auto-import all tasks.
> Edit the fields below to match your schedule, then drag & drop.

## How to use

Each task block follows this format:

```
### Task Name
- name: Task Name
- priority: p0+ | p0 | p1 | p2 | new
- category: thesis | academic | career | platform | life | side
- start: YYYY-MM-DD
- end: YYYY-MM-DD
- hours: total estimated hours (number) or "daily"
- desc: short description of deliverables and subtasks
```

Priority levels:
- **P0+** — hard deadline, non-negotiable
- **P0** — must complete, has a target date
- **P1** — steady progress required, flexible timing
- **P2** — side project, low intensity
- **NEW** — newly added, needs classification

Categories:
- **thesis** — dissertation / thesis work
- **academic** — conferences, reviews, publications
- **career** — job search, visa, professional development
- **platform** — research tools, open-source projects
- **life** — health, admin, driving, personal
- **side** — side projects, explorations
